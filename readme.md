### Prerequisites 项目使用条件

请确保安装以下python第三方库

```.env
[packages]
Django = "*"
djangorestframework = "*"
pillow = "*"
djangorestframework-jwt = "*"
django-filters = "*"
django-filter = "*"
pyyaml = "*"
django-simpleui = "*"

[requires]
python_version = "3.7"
```
